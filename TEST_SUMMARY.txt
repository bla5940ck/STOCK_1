📈 WORKFLOW TEST (工作流測試)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
CSV Download Strategy:
┌─ For each index (^GSPC, ^IXIC, ^SOX):
│  1. Build URL: https://query1.finance.yahoo.com/v7/finance/download/{symbol}
│  2. Add params: period1, period2, interval, events
│  3. Set headers: User-Agent, Accept, Accept-Language
│  4. GET request (10 sec timeout)
│  5. Parse CSV: lines = response.text.split('\n')
│  6. Extract: Date,Open,High,Low,Close,Adj Close,Volume
│  7. Get latest line & extract Open & Close
│  8. Calculate: change_amount = close - open
│  9. Calculate: change_percent = (change_amount / open) * 100
│  10. Create Index object ✅
└─ Aggregate & Cache (5 min TTL)

Fallback Strategy:
┌─ Layer 1: Fresh Cache (5 min TTL) → Success ✅
│  └─ Layer 2: Yahoo Finance CSV → Success ✅
│     └─ Layer 3: Stale Cache (ignore TTL) → Success with warning ✅
│        └─ Layer 4: Error message → Failure ❌
