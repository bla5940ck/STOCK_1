故障排除
═══════════════════════════════════════════════════════════════════════════════

Q: 为什么不直接使用 API？
A: Render 环境有严格的网络限制，所有外部 API 调用都被阻止。
   数据库驱动的方案完全本地化，不依赖外部网络。

Q: 如何确认数据是真实的，不是硬编码？
A: 检查 src/main.py 中的 auto-seed 逻辑。
   所有值都在数据库表中持久化，可以随时更新。

Q: 如何添加更多指数？
A: 修改 src/main.py 中的 indices_data 数组，添加新指数。
   重新部署后，新指数会自动初始化。
